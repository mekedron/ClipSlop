## General

The home tab. Covers:

- **Appearance** — light, dark, or follow system. Opacity and panel size.
- **Launch at login** — keep ClipSlop running automatically.
- **Display mode default** — plain / Markdown / HTML for new prompts.

## Prompts

The prompt library — every built-in prompt and your custom ones, with mnemonic, system prompt, provider override, display mode, and global shortcut bindings. See [Prompt library](./prompt-library.mdx).

## Providers

Configure ChatGPT (OAuth), OpenAI, Anthropic, Ollama, OpenAI-compatible endpoints, and CLI tools. Pick a default and set per-provider temperature / reasoning. See [ChatGPT](./providers/chatgpt.mdx) for the easiest starting point, or [Pick a provider](../pick-a-provider.mdx) for the full menu.

## Keyboard

Remap the four global app shortcuts:

| Default  | Action                              |
|----------|-------------------------------------|
| `⌃⌘C`    | Trigger ClipSlop with selected text |
| `⌃⌘V`    | Process clipboard                   |
| `⌃⌘N`    | Blank editor                        |
| `⇧⌘2`    | Screen capture (OCR)                |

Per-prompt shortcuts live in the **Prompts** tab. See [Keyboard shortcuts](./keyboard-shortcuts.md) for the full customisation guide.

## iCloud Sync

One toggle — when enabled, your prompt library mirrors across all Macs signed into the same iCloud account. See [iCloud sync](./icloud-sync.mdx).
