# Built-in prompts

ClipSlop ships with a curated catalogue. Six top-level groups, each with a single-letter mnemonic.

```
[‚åò/]  // Your prompt ‚Äî type // followed by your instruction to run a one-off custom prompt
[T]   Translate   ‚Üí English, Finnish, Russian, Spanish, French, German, + 12 more
[R]   Rewrite     ‚Üí Elaborate, Neutral, Professional, Warm, Business, Playful, Biblical
[F]   Format      ‚Üí Fix Grammar, Clean Up, Beautify Code, Reformat, Email, Markdownify, HTMLify
[D]   Dev         ‚Üí Add Comments, Beautify Code, Clean Logs, Explain Code, Explain Stack Trace, Naming
[A]   Analyze     ‚Üí Summary, Explain Simply, TL;DR, Condense 20%
[C]   Convert     ‚Üí HTML, Markdown
```

## Translate
18 languages out of the box: English, Finnish, Russian, Spanish, French, German, Italian, Portuguese, Dutch, Swedish, Norwegian, Danish, Polish, Japanese, Korean, Mandarin, Arabic, Turkish.

Each language has its own mnemonic ‚Äî `T` ‚Üí `E` is English, `T` ‚Üí `F` is Finnish, and so on. Open **Settings ‚Üí Prompts ‚Üí Translate** to see the full mapping or rebind keys.

## Rewrite
Seven tones for the same input:

- **Elaborate** ‚Äî expand a terse note into a full paragraph.
- **Neutral** ‚Äî strip emotional or persuasive language.
- **Professional** ‚Äî formal but readable. Default for business writing.
- **Warm** ‚Äî friendlier than Neutral, less formal than Professional.
- **Business** ‚Äî corporate-toned; terse, action-oriented.
- **Playful** ‚Äî informal, lighter, punchier.
- **Biblical** ‚Äî for the lulz.

## Format
Seven tools that operate on the surface of the text:

- **Fix Grammar** ‚Äî corrects grammar, spelling, punctuation. Quick Paste default <kbd>‚åÉ‚åòG</kbd>.
- **Clean Up** ‚Äî removes line-noise (extra whitespace, broken markup, line numbers).
- **Beautify Code** ‚Äî passes code through a formatter prompt for the language guessed.
- **Reformat** ‚Äî reflows for the destination context. Quick Paste default <kbd>‚åÉ‚åòF</kbd>.
- **Email** ‚Äî turns notes into a polite email body.
- **Markdownify** ‚Äî converts plain text or HTML to Markdown.
- **HTMLify** ‚Äî the inverse ‚Äî Markdown or plain text to HTML.

## Dev
For developers:

- **Add Comments** ‚Äî annotates code with inline comments.
- **Beautify Code** ‚Äî same as Format ‚Üí Beautify Code, surfaced in the Dev folder for muscle memory.
- **Clean Logs** ‚Äî strips timestamps, ANSI codes, and noise from logs.
- **Explain Code** ‚Äî line-by-line plain-English walk-through.
- **Explain Stack Trace** ‚Äî finds the cause and reframes the trace in human terms.
- **Naming** ‚Äî suggests better names for variables / functions / types.

## Analyze
For incoming text:

- **Summary** ‚Äî a paragraph or bullet list, depends on input length.
- **Explain Simply** ‚Äî ELI5. Open &amp; Run default <kbd>‚åÉ‚åò‚å•A</kbd>.
- **TL;DR** ‚Äî one-sentence compression.
- **Condense 20%** ‚Äî keeps the structure, trims fluff.

## Convert
- **HTML** ‚Äî Markdown (or plain) ‚Üí HTML. Strips presentational cruft.
- **Markdown** ‚Äî HTML ‚Üí Markdown. Preserves headings, lists, code blocks.

## Default global shortcuts
Some prompts ship with a default global hotkey. **Quick Paste** runs the prompt inline; **Open &amp; Run** opens ClipSlop with the result ready.

| Shortcut    | Prompt                | Mode        |
|-------------|-----------------------|-------------|
| `‚åÉ‚åò/`       | // Your prompt        | Quick Paste |
| `‚åÉ‚åò‚å•/`      | // Your prompt        | Open &amp; Run  |
| `‚åÉ‚åòG`       | Fix Grammar           | Quick Paste |
| `‚åÉ‚åòF`       | Reformat              | Quick Paste |
| `‚åÉ‚åòT`       | Translate ‚Üí English   | Quick Paste |
| `‚åÉ‚åò‚å•T`      | Translate ‚Üí English   | Open &amp; Run  |
| `‚åÉ‚åò‚å•A`      | Explain Simply        | Open &amp; Run  |

All of these can be reassigned, removed, or added to other prompts in **Settings ‚Üí Prompts**.
