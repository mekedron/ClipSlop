# Run in Editor

Same idea as [Inline Run](./quick-paste.mdx), but ClipSlop opens its panel with the result ready, so you can review, tweak, or keep chaining. Useful for translations and longer rewrites where you want a chance to review before pasting.

## Default Run in Editor shortcuts

| Shortcut  | Prompt              |
|-----------|---------------------|
| `⌃⌘⌥/`    | // Your prompt      |
| `⌃⌘⌥T`    | Translate → English |
| `⌃⌘⌥A`    | Explain Simply      |

Run in Editor pairs with [the full pipeline](./full-pipeline.mdx) — once the panel is open, you can chain prompts, branch from history, or copy the result manually with <kbd>⌘C</kbd>.

## When to use each

| Situation                                    | Use         |
|----------------------------------------------|-------------|
| One-shot fix you trust (grammar, formatting) | Inline Run |
| Translation you might want to tweak          | Run in Editor  |
| Multiple prompts in sequence                 | Run in Editor  |
| Long-form rewriting with branching           | Run in Editor  |

Bind your own per-prompt shortcuts in **Settings → Prompts**.
